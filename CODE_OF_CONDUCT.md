# Contributor Covenant Code of Conduct

We as members, contributors, and leaders pledge to make participation in our community a
harassment-free experience for everyone.

We pledge to act and interact in ways that contribute to an open, welcoming, diverse,
inclusive, and healthy community. Examples of unacceptable behavior include harassment,
trolling, insulting/derogatory comments, and publishing others' private information.

Project maintainers are responsible for clarifying standards and may remove contributions
or contributors that violate this Code of Conduct.

Report concerns to **conduct@everjust.co**.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.
