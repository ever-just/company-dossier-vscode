# Security Policy

## Reporting a vulnerability
Please email **security@everjust.co** with details and reproduction steps. We aim to
acknowledge within a few business days. Please do not open public issues for sensitive
vulnerabilities.

## Scope & data handling
- Company Dossier uses **public sources only** — no authentication bypass, no private-data
  harvesting, no social engineering.
- Any API keys you configure (e.g. for AI synthesis) stay in your VS Code settings / local
  environment and are sent only to the provider you configured. They are never transmitted
  to EVERJUST.
- The extension writes files into your workspace; review generated output before sharing.

## Supported versions
The latest published version receives security fixes.
